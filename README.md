# QT-C-Gym_Reservation_Platfrom

## 项目介绍

这是一个**基于QT和MySQL的体育馆预约平台**

**项目介绍**：该平台分为**客户端**和**管理员端**，为用户提供体育馆预约平台，为管理员提供用户和体育馆信息管理平台。

**功能**：
1. 用户端有注册、登录、个人信息修改、预约体育场地功能；
2. 管理员端有对体育场信息和顾客信息管理功能，实现信息的增删改查功能；
